# 11-02: Add CI workflow and audit command surface

## Goal

GitHub Actions can run the project verification story against a Postgres service, while audit commands are available but non-blocking until cleanup is complete.

## Tasks

- Add `.github/workflows/ci.yml`.
- Use Corepack/Yarn immutable install.
- Start Postgres 16 service and run Drizzle migrations.
- Run typecheck, lint, API memory tests, API DB tests, web pet tests, full test, and build.
- Add non-blocking dependency, circular dependency, and dead-code audit steps.

## Verification

- `yarn typecheck`
- `yarn lint`
- `yarn test`
- YAML is parseable by GitHub Actions syntax conventions.
